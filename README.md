# How to run the App

1. Clone or pull updates
    Clone - `git clone https://github.com/victoriaowolijah/shopify.git`
    Pull -  `git pull origin <branch>`

2. Install dependencies
    `yarn install` if you do not have yarn RUN: `npm install yarn -g`

3. Run the app 
    `yarn start`
